# 📘 Statistics & Probability — Quick Revision Notes

![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-success)
![License](https://img.shields.io/badge/License-Educational-blue)
![HTML](https://img.shields.io/badge/Built%20With-HTML%2FCSS%2FJS-orange)
![Responsive](https://img.shields.io/badge/Responsive-Yes-brightgreen)

A collection of concise **Statistics & Probability** revision notes created by **Dr. Ronald Bbosa** and published with permission.

This project transforms traditional revision sheets into an interactive website, making it easier for students to quickly review essential concepts before coursework, assignments, quizzes, and examinations.

## 🌐 Live Website

➡️ **https://manthank1.github.io/stats-probability-quick-notes/**

---

# ✨ Features

- 📖 Clean, responsive interface
- 🔍 Full-screen image lightbox
- ⌨️ Keyboard navigation (← → Esc)
- 📥 Download individual revision notes
- 📦 Download all notes as a single ZIP file
- 📱 Mobile-friendly design
- ⚡ Lightweight (HTML, CSS & JavaScript only)
- 🚀 No frameworks or build tools required

---

# 📚 Topics Covered

## Fundamentals

- Types of Statistics
- Types of Data
- Measurement Scales
- Box Plot

## Probability

- Central Limit Theorem (CLT)
- Bayes' Theorem

## Hypothesis Testing

- Six-Step Hypothesis Testing Workflow
- One-Tailed vs Two-Tailed Tests
- One-Sample Z-Test
- One-Sample t-Test
- Independent Samples t-Test
- Paired Samples t-Test
- Worked Examples
- Practice Questions

## ANOVA

- One-Way ANOVA
- Repeated Measures ANOVA

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/manthanK1/stats-probability-quick-notes.git
```

Navigate into the project

```bash
cd stats-probability-quick-notes
```

Open `index.html` in your preferred web browser.

No installation, package manager, or build process is required.
